# 名词术语表

本文描述项目所用的藏经专业名词及其变量命名约定。

| 变量命名   |  中文名词  |  含义       |
|-----------|:-------:|-------------------:|
| canon     | 藏经    | 例如嘉兴藏、永乐北藏   |
| tripitaka | 三藏典籍 | 经论体系，与canon同义 |
| canon_code | 藏经编码 | 例如 T、GA |
| sutra    | 经文    | 具体的某篇经论文章 |
| sutra_code | 经号  | 例如 GA0008  |
| category | 部类    | 例如阿含部、瑜伽部 |
